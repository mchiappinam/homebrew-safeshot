# Homebrew Tap for SafeShot

Privacy-first screenshot tool. No cloud, no tracking, no compromises.

## Install

```
brew tap mchiappinam/safeshot
brew install --cask safeshot
```

## Update

```
brew upgrade --cask safeshot
```

## Uninstall

```
brew uninstall --cask safeshot
```

## About

SafeShot is a lightweight screenshot tool that lives in your system tray. Capture, annotate, and save. Everything stays on your machine.

For more info, see the [main repo](https://github.com/mchiappinam/SafeShot).
